SkyGuard-X - Enterprise Drone Defense System

![Drone Defense](https://via.placeholder.com/800x400/0f1f3a/ffffff?text=SkyGuard-X+Enterprise+Dashboard)

Advanced AI-powered drone detection and neutralization system with real-time enterprise dashboard for military, government, and critical infrastructure protection.

Features
- Real-time Monitoring -  Live drone detection and tracking
- AI-Powered Analytics - Advanced threat detection algorithms  
- Interactive Dashboard - Professional web interface with dark/light mode
- Alert System - Multi-level notifications with sound alerts
- Live Maps - OpenStreetMap integration for real-time tracking
- Responsive Design - Works on desktop, tablet, and mobile

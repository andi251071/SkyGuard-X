# ?? SkyGuard-X Deployment Guide

## System Requirements
- Python 3.8 or higher
- 4GB RAM minimum
- Web browser with JavaScript enabled

## Quick Deployment

### Windows
1. Clone repository:
   ```bash
   git clone https://github.com/andi251071/SkyGuard-X.git
   cd SkyGuard-X
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Launch system:
   ```bash
   python run.py
   ```
   OR double-click: `launch.bat`

4. Access dashboard:
   - Automatically opens: http://localhost:8000/dashboard/index.html
   - Manual: Open browser to above URL
